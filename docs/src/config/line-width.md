# lineWidth

Fallback line width. Inherited from the top-level `lineWidth` in dprint.json when unset

|             |                                                             |
| ----------- | ----------------------------------------------------------- |
| **Type**    | `integer`                                                   |
| **Default** | *inherits from the top-level key in the same `dprint.json`* |

## Config

```json
{
  "svg": {
    "lineWidth": null
  }
}
```
