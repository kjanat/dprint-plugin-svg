# useTabs

Use tabs for indentation. Inherited from the top-level `useTabs` in dprint.json when unset

|             |                                                             |
| ----------- | ----------------------------------------------------------- |
| **Type**    | `boolean`                                                   |
| **Default** | *inherits from the top-level key in the same `dprint.json`* |

## Config

```json
{
  "svg": {
    "useTabs": null
  }
}
```
