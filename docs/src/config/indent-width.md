# indentWidth

Indent width when useTabs is false. Inherited from the top-level `indentWidth` in dprint.json when unset

|             |                                                             |
| ----------- | ----------------------------------------------------------- |
| **Type**    | `integer`                                                   |
| **Default** | *inherits from the top-level key in the same `dprint.json`* |

## Config

```json
{
  "svg": {
    "indentWidth": null
  }
}
```
