# indentWidth

Indent width when useTabs is false. Inherited from dprint global indentWidth when unset

|             |                                       |
| ----------- | ------------------------------------- |
| **Type**    | `integer`                             |
| **Default** | *inherited from dprint global config* |

## Config

```json
{
  "svg": {
    "indentWidth": null
  }
}
```
